## 🌸 About

Riya's Crochet Corner is a modern crochet marketplace where customers can explore and purchase handmade crochet creations including:

- 💐 Crochet Bouquets
- 🌷 Crochet Flowers
- 🔑 Keychains
- 🧸 Handmade Accessories

The website provides a smooth shopping experience with product browsing, order management, customer reviews, and secure Firebase integration.

---

## ✨ Features

- 🛍️ Product Catalog
- 🔍 Product Search & Filtering
- ⭐ Customer Reviews
- 📦 Order Management
- 📸 Payment Screenshot Upload
- 🔐 Firebase Authentication
- ☁️ Cloudinary Image Storage
- 📱 Responsive Design
- 👩‍💼 Admin Dashboard
